# 小说素材收集 - 安装常见问题

---

## macOS 常见问题

### 问题 1：提示"已损坏，无法打开"

这是 macOS 最常见的安全提示，按以下步骤解决：

**方法一：使用终端命令（推荐，最有效）**

1. 打开 **终端**（按 `Command + 空格`，输入"终端"，回车）
2. 复制以下命令，粘贴到终端中，按回车：

```bash
xattr -cr /Applications/小说素材收集.app
```

3. 再次打开应用，即可正常使用

**方法二：右键打开**

1. 在 Finder（访达）中找到 `小说素材收集.app`
2. **不要双击，右键点击**它
3. 在菜单中选择"打开"
4. 会弹出一个对话框，点击"打开"按钮
5. 以后就可以正常双击打开了

---

### 问题 2：提示"无法验证开发者"

**解决方法：**

1. 打开 **系统设置**（或系统偏好设置）
2. 点击左侧的 **隐私与安全性**
3. 向下滚动，找到"安全性"部分
4. 会看到提示："小说素材收集.app"已被阻止，点击 **仍要打开** 按钮
5. 输入密码确认即可

---

### 问题 3：DMG 打开后无法拖拽到应用程序

**解决方法：**

1. 直接把 `小说素材收集.app` 拖到桌面上
2. 然后从桌面拖到 **应用程序** 文件夹
3. 再按上面的方法解决"已损坏"问题

---

### 问题 4：我是 Intel Mac 还是 Apple Silicon Mac？

- **Apple Silicon Mac**（M1/M2/M3/M4 芯片）：下载 `aarch64` 版本
- **Intel Mac**：下载 `x64` 版本

**如何查看：**
1. 点击左上角苹果图标
2. 选择"关于本机"
3. 如果看到"M1"、"M2"、"M3"、"M4"等字样，就是 Apple Silicon
4. 如果看到"Intel"字样，就是 Intel Mac

## Windows 常见问题

### 问题 1：安装时提示"Windows 已保护你的电脑"

**解决方法：**

1. 弹窗上点击"更多信息"
2. 会显示应用信息，点击"仍要运行"
3. 再点击"运行"确认

---

### 问题 2：杀毒软件报警

**原因：** 未签名的应用可能被误报为风险。

**解决方法：**

**360 安全卫士：**
- 在弹窗中选择"允许"或"添加到信任区"

**腾讯电脑管家：**
- 点击"允许"或"恢复文件"

**Windows Defender：**
- 点击"允许"或"在设备上允许"

---

### 问题 3：双击安装包没反应

**可能的原因：**
- 被杀毒软件后台拦截了
- 文件下载不完整

**解决方法：**
1. 先检查杀毒软件的拦截记录，选择"允许"
2. 重新下载安装包
3. 下载完成后右键安装包，选择"以管理员身份运行"

---

### 问题 4：提示需要安装 WebView2

应用需要 WebView2 运行环境，Windows 10/11 通常已内置。如果提示缺少：

1. 访问微软官网：https://developer.microsoft.com/en-us/microsoft-edge/webview2/
2. 下载"Evergreen Bootstrapper"版本
3. 安装后重新运行应用安装程序

## 数据存储位置

你的所有数据（素材库、拆文、笔记等）都存储在本地，不会上传到网络：

- **macOS：** `~/Library/Application Support/小说素材收集/`
- **Windows：** `%APPDATA%/小说素材收集/`

**备份数据：** 复制上述文件夹到安全位置即可。

**升级/重装：** 数据会自动保留，不会丢失。

---

## 如何更新应用

1. 打开应用
2. 点击左下角的版本号
3. 点击"检查更新"
4. 如果有新版本，按提示下载安装即可

**注意：** 更新不会删除你已有的数据。

## 如何卸载

### macOS 卸载方法

1. 打开 Finder（访达）
2. 点击左侧"应用程序"
3. 找到"小说素材收集.app"
4. 右键点击 → 移到废纸篓
5. 清空废纸篓

### Windows 卸载方法

1. 打开"设置" → "应用" → "已安装的应用"
2. 找到"小说素材收集"
3. 点击 → 卸载
