## macOS 本地编译

### 前置要求
- Node.js 20+
- Rust (通过 `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh` 安装)
- Xcode Command Line Tools (`xcode-select --install`)

### 编译步骤
```bash
# 安装依赖
npm install

# 开发模式运行
npm run tauri dev

# 编译生产版本
npm run tauri build
```

编译后的文件位于：`src-tauri/target/release/bundle/dmg/`

## Windows 编译方案

### 方案一：GitHub Actions 自动编译（推荐）

**优点**：
- 无需 Windows 电脑
- 每次打 tag 自动编译 macOS 和 Windows 两个版本
- 自动创建 GitHub Release

**使用步骤**：

1. **推送代码到 GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/novel-material-collector.git
   git push -u origin main
   ```

2. **创建标签触发编译**
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```

3. **等待编译完成**
   - 访问 GitHub 仓库的 "Actions" 页面
   - 等待编译完成（约 10-15 分钟）

4. **下载安装包**
   - 编译完成后，在 "Releases" 页面可以看到自动创建的 release
   - 下载 Windows 版本的 `.msi` 或 `.exe` 文件

---

### 方案二：Windows 本地编译

**前置要求**：
- Windows 10/11
- Node.js 20+
- Rust (通过 https://rustup.rs 安装)
- Visual Studio Build Tools (C++ 开发工具)

**编译步骤**：
```powershell
# 安装依赖
npm install

# 编译
npm run tauri build
```

编译后的文件位于：`src-tauri/target/release/bundle/msi/`

---

### 方案三：借用 Windows 电脑

如果你有朋友用 Windows：
1. 把整个项目文件夹打包发给他
2. 让他安装 Node.js 和 Rust
3. 运行 `npm install` 和 `npm run tauri build`
4. 把编译好的 `.msi` 文件发给你
