## 📦 下载安装包

### macOS 用户
- **文件名**：`扫榜助手_1.0.0_aarch64.dmg`
- **大小**：3.5 MB
- **支持系统**：macOS（Apple Silicon M1/M2/M3 芯片）

- **文件名**：`扫榜助手_1.0.0_x64-setup.exe` 或 `扫榜助手_1.0.0_x64_en-US.msi`

- **安装方式**：双击 `.exe` 或 `.msi` 文件安装

## 🚀 如何获取 Windows 版本

### 方式一：从 GitHub Releases 下载（推荐）
如果你看到这个文档，说明 macOS 版本已经编译完成。Windows 版本需要通过 GitHub Actions 自动编译：

1. **项目维护者操作**：
   ```bash
   # 推送代码到 GitHub
   git init
   git add .
   git commit -m "v1.0.0 release"
   git branch -M main
   git remote add origin https://github.com/ishiningx/novel-material-collector.git
   git push -u origin main

   # 创建标签触发编译
   git tag v1.0.0
   git push origin v1.0.0
   ```

2. **等待编译**（约 10-15 分钟）
   - 访问 GitHub 仓库的 "Actions" 页面
   - 等待 "Build and Release" 工作流完成

3. **下载安装包**
   - 在 "Releases" 页面找到 v1.0.0
   - 下载 `扫榜助手_1.0.0_x64-setup.exe` 或 `.msi` 文件

### 方式二：本地编译（开发者）
如果你有 Windows 电脑和开发环境：

**前置要求**：
- Node.js 20+
- Rust (https://rustup.rs)
- Visual Studio Build Tools (C++ 开发工具)

**编译步骤**：
```powershell
git clone https://github.com/ishiningx/novel-material-collector.git
cd novel-material-collector
npm install
npm run tauri build
```

编译后的文件位于：`src-tauri/target/release/bundle/msi/`
