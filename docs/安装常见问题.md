# 扫榜助手 - 安装常见问题

---

## macOS 常见问题

### 问题 1：提示"无法打开，因为无法验证开发者"

**原因**：macOS 安全机制，第一次打开会拦截。

**解决方法（最简单）**：

1. 在 Finder（访达）中找到 `扫榜助手.app`
2. **不要双击，右键点击**它
3. 在菜单中选择"打开"
4. 会弹出一个对话框，点击"打开"按钮
5. 以后就可以正常双击打开了

---

### 问题 2：提示"文件已损坏，无法打开"

**原因**：macOS 安全检查过于严格。

**解决方法**：

1. 打开 **系统设置**（或系统偏好设置）
2. 点击左侧的 **隐私与安全性**
3. 向下滚动，找到"安全性"部分
4. 会看到提示："扫榜助手.app"已被阻止，点击 **仍要打开** 按钮
5. 输入密码确认即可

---

### 问题 3：DMG 打开后无法拖拽到 Applications

**原因**：可能是权限问题。

**解决方法**：

1. 直接把 `扫榜助手.app` 拖到桌面上
2. 然后从桌面拖到 **应用程序** 文件夹
3. 或者：在应用程序文件夹里找到它，双击打开

---

## Windows 常见问题

### 问题 1：安装时提示"Windows 已保护你的电脑"

**原因**：Windows SmartScreen 安全功能。

**解决方法**：

1. 弹窗上点击"更多信息"
2. 会显示应用信息，点击"仍要运行"
3. 再点击"运行"确认

---

### 问题 2：杀毒软件报警

**原因**：未签名的应用可能被误报为风险。

**解决方法**：

**360 安全卫士**：
- 在弹窗中选择"允许"或"添加到信任区"

**腾讯电脑管家**：
- 点击"允许"或"恢复文件"

**Windows Defender**：
- 点击"允许"或"在设备上允许"

---

### 问题 3：双击安装包没反应

**可能的原因**：
- 被杀毒软件后台拦截了
- 文件下载不完整

**解决方法**：
1. 先检查杀毒软件的拦截记录，选择"允许"
2. 重新下载安装包
3. 下载完成后右键安装包，选择"以管理员身份运行"

---

## 数据存储位置

你的所有数据（素材库、分类标签等）都存储在本地，不会上传到网络：

- **数据位置**：应用内部自动管理
- **升级不会丢失数据**
- **卸载后数据会保留**（如果想彻底删除，需要手动清理）

---

## 如何卸载

### macOS 卸载方法
1. 打开 Finder（访达）
2. 点击左侧"应用程序"
3. 找到"扫榜助手.app"
4. 右键点击 → 移到废纸篓
5. 清空废纸篓

### Windows 卸载方法
1. 打开"控制面板"
2. 点击"程序" → "程序和功能"
3. 找到"扫榜助手"
4. 右键点击 → 卸载
